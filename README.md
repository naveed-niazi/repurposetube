# RepurposeTube

**Turn any YouTube video into 6 content formats instantly.**

Paste a YouTube URL and RepurposeTube generates — in parallel — a blog post, Twitter/X thread, LinkedIn post, YouTube SEO package (A/B titles, description, 15 tags), newsletter email, and 3 Shorts scripts.

**Domain**: [repurposetube.com](https://repurposetube.com)

## Tech Stack

- **Framework**: Next.js (full-stack)
- **Database / Auth / Storage**: Supabase + Prisma
- **Background Jobs**: Inngest
- **UI**: shadcn/ui + Radix UI
- **Hosting**: Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3003](http://localhost:3003) to view the app.
